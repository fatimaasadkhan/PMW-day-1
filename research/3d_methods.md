# 3D Reconstruction Methods

## COLMAP

Creates accurate 3D models from multiple photographs using Structure from Motion (SfM). It is widely used for heritage documentation because it produces detailed point clouds and camera positions.

## NeRF (Neural Radiance Fields)

Uses deep learning to reconstruct realistic 3D scenes from multiple images. It generates high-quality novel views but requires a powerful GPU and longer processing time.

## Gaussian Splatting

A modern 3D rendering technique that represents scenes using thousands of Gaussian points. It is faster than NeRF while maintaining excellent visual quality.
